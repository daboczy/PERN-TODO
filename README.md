# TODO app
> Full Stack Node, Vue, Express, Postgres

> rewrite from react to vue

## Quick Start
```bash
# Install dependencies
npm install

# Start Express Server: http://localhost:5000
cd server
npm run start

# Start Vue DevServer: http://localhost:8080
cd client
npm run serve

# Build for production
cd client
npm run build
```
